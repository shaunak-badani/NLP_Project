# NLP_Project

> Project for AIPI 540: Deep Learning and Applications for the NLP module.


### How to run

- Frontend
```
cd frontend
npm install
npm run dev
```
- The application is then hosted on `localhost:5173`


- Backend
```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
fastapi dev main.py
```
